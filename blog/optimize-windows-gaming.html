How to Optimize Windows for Gaming

January 2, 2026 · 7 min read · By Tech Team

You don't need expensive software to optimize Windows for gaming. These free built-in settings can noticeably improve performance, reduce input lag, and eliminate stuttering.

Note: These tips work for Windows 10 and 11. Create a system restore point before making changes.

1. Enable Game Mode

Windows Game Mode

Game Mode prioritizes system resources for games and prevents Windows Update from installing drivers mid-game.

Settings → Gaming → Game Mode → On

2. Disable Hardware-Accelerated GPU Scheduling (Test Both)

GPU Scheduling

This feature can help or hurt depending on your hardware. Test with it on and off.

Settings → System → Display → Graphics → Change default graphics settings

Toggle "Hardware-accelerated GPU scheduling" and test your games.

3. Set High Performance Power Plan

Power Plan

Balanced mode throttles CPU/GPU to save power. High Performance removes these limits.

Control Panel → Power Options → High Performance

If you don't see High Performance, click "Create a power plan" or run:

powercfg -duplicatescheme 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c

4. Disable Fullscreen Optimizations

For Each Game

Windows' fullscreen optimizations can add input lag. Disable per-game:

  1. Right-click game .exe → Properties
  2. Compatibility tab
  3. Check "Disable fullscreen optimizations"
  4. Click Apply

5. Update Graphics Drivers

Get Latest Drivers

Don't use Windows Update for GPU drivers. Get them direct from:

  • NVIDIA: GeForce Experience or nvidia.com
  • AMD: AMD Adrenalin or amd.com
  • Intel: Intel Driver Support Assistant

6. Disable Background Apps

Stop Background Bloat

Windows 11:

Settings → Apps → Installed apps → [App] → Advanced options → Background permissions: Never

Windows 10:

Settings → Privacy → Background apps → Toggle off apps you don't need

7. Disable Xbox Game Bar

Remove Overlay Overhead

If you don't use it, disable it:

Settings → Gaming → Xbox Game Bar → Off Settings → Gaming → Captures → Record in background → Off

8. Set GPU Preference Per Game

Force Dedicated GPU

Ensure games use your dedicated GPU, not integrated graphics:

Settings → System → Display → Graphics

Add your game and set to "High performance"

9. Disable Startup Programs

Clean Boot

Fewer programs running = more resources for games:

Ctrl+Shift+Esc → Startup tab → Disable unnecessary apps

10. Enable Ultimate Performance (Desktop Only)

Hidden Power Plan

More aggressive than High Performance. Run in admin PowerShell:

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

Then select "Ultimate Performance" in Power Options.

Warning: Ultimate Performance uses more power and generates more heat. Only use on desktops with good cooling.

What Won't Help

  • "Gaming" RAM cleaner software – Windows manages RAM fine
  • Registry "tweaks" – Risk outweighs minimal benefit
  • Disabling Windows services – Can cause instability
  • Third-party "optimization" software – Usually bloatware

The Biggest Impact

Honestly? Hardware matters most. If you're still struggling after these tweaks:

  • Add more RAM (16GB minimum for modern games)
  • Upgrade to an SSD if you're on a hard drive
  • Consider a GPU upgrade
